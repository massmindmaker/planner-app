# Пользовательские воркфлоу — Бесконечный Планер

## 1. Первый запуск
- Пользователь открывает приложение → видит пустой Dashboard
- Energy prompt: "Как твоя энергия?" → выбор 1-5 эмодзи
- Sidebar содержит навигацию ко всем разделам
- Рекомендуемый путь: Миссия → Цели на год → Привычки на месяц

## 2. Цели на жизнь (/life)
- 6 карточек по категориям: Карьера, Финансы, Творчество, Личное развитие, Взаимоотношения, Духовность
- "Добавить цель" → ввод названия и описания
- Inline-editing: клик по тексту → редактирование на месте
- Удаление: кнопка X с подтверждением

## 3. Годовые цели (/goals)
- 6 карточек по категориям, до 8 целей в каждой
- Квартальная тема сверху (InlineEdit)
- Каждая цель: чекбокс завершения + название + поля привычек (дневная/недельная/месячная)
- Можно привязать к цели на жизнь (lifeGoalId)
- Skeleton loading при загрузке

## 4. Месячный фокус (внутри /month/N)
- Секция "Фокус месяца" вверху страницы месяца
- Добавление задач на месяц (3-5 рекомендуется)
- Чекбокс завершения + inline-editing названия
- Удаление с undo-toast (5 сек на отмену)
- Прогресс-бар показывает % выполнения

## 5. Ежедневные привычки (/month/N)
### Создание привычки
- Кнопка "+" → диалог создания:
  - Название
  - Полярность: Позитивная (отмечаешь когда ДЕЛАЛ) / Негативная (отмечаешь когда ПРОВАЛИЛСЯ)
  - Тип: Да/Нет | Числовая | X раз в неделю
  - Для числовой: цель + единица измерения
  - Для X раз в неделю: количество раз
  - Минимальная версия (опционально)

### Трекинг
- Таблица: строки = привычки, столбцы = дни месяца
- Позитивные: зелёный чекбокс = сделал
- Негативные: красный X = провалился, пустые дни = успех
- Числовые: клик → popover с вводом числа, ячейка показывает "5/8"
- Context menu (правый клик): Выполнено / Минимум (жёлтый) / Не выполнено
- Drag-to-reorder: перетаскивание привычек для изменения порядка
- Стрик-бейдж: 🔥12 для позитивных, ✓45д для негативных

### Прогресс
- Строка прогресса внизу: % за каждый день (цветовая шкала)
- Общий прогресс месяца в шапке

## 6. Еженедельные привычки (/month/N)
- Секция под ежедневными привычками
- Таблица: строки = привычки, столбцы = Неделя 1-5
- Чекбокс на каждую неделю

## 7. Рутины (/routines)
- Карточки рутин: Утренняя / Вечерняя / Своя
- "Создать рутину" → диалог с названием и типом
- Внутри рутины: упорядоченный список привычек
- Drag-to-reorder привычек (dnd-kit)
- Timeline визуализация: вертикальная линия с узлами привычек
- Бонус XP +20 за выполнение всей рутины

## 8. Еженедельный планировщик (/week/N)
- Номер недели + диапазон дат (Пн-Вс)
- Навигация: ← → между неделями
- "Фокус недели" — inline-editable текст
- Сводка привычек за неделю (Пн-Вс таблица)
- Energy strip: эмодзи энергии за каждый день
- Кнопка "Написать обзор"

## 9. Weekly Review
- Автоматическая статистика: выполнено привычек, % прогресса
- Структурированные вопросы:
  - "Что прошло хорошо?"
  - "Что не получилось?"
  - "Фокус на следующую неделю?"
- Свободное поле "Заметки"
- Рейтинг недели: 1-5 звёзд
- Сохранение: +30 XP

## 10. Monthly Review (из /month/N)
- Кнопка "Итоги месяца" → модальное окно
- Те же вопросы что и Weekly Review
- Автостатистика за месяц
- Сохранение: +50 XP

## 11. Колесо жизни (/wheel)
- Radar chart с 6 осями (категории)
- Выбор месяца (12 кнопок)
- Ввод оценок 1-10 по каждой категории (слайдеры)
- Сохранение → +20 XP
- Наложение: текущий месяц vs предыдущий
- История: линейный график по месяцам

## 12. Аналитика (/analytics)
- Тепловая карта (heatmap): 52 недели × 7 дней, цвет = % выполнения
- Тренды: линейный график % по неделям
- Лучшие дни: столбчатая диаграмма по дням недели
- Сравнение месяцев: текущий vs предыдущий
- Корреляция энергии: scatter plot
- Рейтинг привычек: таблица по % выполнения
- Топ стриков: лидерборд

## 14. Энергетический трекер
- На Dashboard: "Как твоя энергия?" с 5 эмодзи (😴😐🙂😊🔥)
- Одна оценка в день
- Отображается на странице недели
- Корреляция с привычками в аналитике
- +5 XP за запись

## 15. Квартальные темы
- Баннер на Dashboard и странице целей
- Inline-editing: клик → редактирование
- "Задай тему квартала" если не задана

## 16. Dark Mode
- Переключатель в sidebar (Sun/Moon иконка)
- Сохраняется в localStorage
- Автодетект системных настроек

## 17. Переключатель года
- Dropdown в sidebar: прошлый / текущий / следующий год
- Все данные переключаются по годам
- Сохраняется в localStorage

## 18. Горячие клавиши (в month tracker)
- Space/Enter — переключить привычку
- Стрелки — навигация по ячейкам
- N — новая привычка
- ? — показать список клавиш
